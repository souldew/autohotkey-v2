# AutoHotkey v2

| 修飾キー | 表示 |
| -------- | ---- |
| Ctrl     | ^    |
| Shift    | +    |
| Windows  | #    |
| Alt      | +    |

| 入力キー       | 表示 |
| -------------- | ---- |
| 変換(右)       | vk1C |
| 無変換(左)     | vk1D |
| ; (セミコロン) | vkBB |
| : (コロン)     | vkBA |

# Powershell で kill

Stop-process -name "AutoHotkeyV2"
